# AWS 기반 온라인 서점 취약점 진단 - 프로젝트 산출물

SK쉴더스 루키즈 30기 생성형 AI 활용 사이버보안 전문인력 양성과정 3조 팀프로젝트 산출물입니다.
AWS 기반 온라인 서점(CheckBang) 환경을 직접 구축하고, Windows/Unix/DBMS/WAS/Web Server/AWS 클라우드 영역에 대해
최초 진단 → 이행 조치 → 이행 점검까지 전체 취약점 진단 프로세스를 수행했습니다.

관련 코드 저장소:
- [ai-vuln-scanner](https://github.com/WG-K/ai-vuln-scanner) - AI 활용 취약점 진단 대시보드
- [auto-vuln-scanner](https://github.com/WG-K/auto-vuln-scanner) - 자동화 취약점 진단 스크립트

## 폴더 구성

| 폴더 | 내용 |
|---|---|
| `1_수행계획서` | 웹/인프라/클라우드 취약점진단 수행계획서 |
| `2_최초진단결과보고서` | Windows/Unix/DBMS/WAS/Webserver/Web(user,admin)/클라우드 최초 진단 결과보고서 |
| `3_이행점검보고서` | 조치 이행 여부에 대한 이행점검보고서 |
| `4_AI진단산출물` | 자동화 스크립트 Raw Data를 AI로 분석한 진단 결과(초기/최종) |
| `WBS.xlsx` | 프로젝트 일정 관리표 |
| `리소스 사용계획서` | 클라우드 리소스 사용계획서 |

## 참고

- 팀 공동 산출물이며, 개인 포트폴리오 목적으로 공유합니다.
- 진단 대상 AWS 계정 정보 등 민감 정보는 마스킹 처리했습니다.
